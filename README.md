CAPTURE Scale is an online tool for projecting HIV and TB intervention costs at scale, given capacity constraints and possible economies of scope. 

## Developing
Requirements:
* npm

## Running locally
* clone this repo
* from within this directory: `npm install`
* `npm run dev` will start a local server

## Deploying
`npm run build` uses Vite to bundle the app into the `dist` folder. This will be run using GitHub actions and deployed to GitHub pages.
